# nodejs-workbench
2017-04-07 weschen，一个使用Nodejs写的工作后台应用，使用的是以下技术
* Nodejs
* MongoDB
* Express \ Jade \ Bower
* Gulp
* Yoman

## 在开始之前
```
npm install -g yo
npm install -g bower
npm install -g gulp
npm install -g generator-express
```

## 创建项目
```
yo express
```

## 克隆项目后安装依赖
```
npm install
```

## 运行项目
```
gulp
```
